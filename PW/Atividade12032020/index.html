As tags html são tags utilizadas para a composição da sua pagina, cada tag tem sua função, veja a seguir uma explicação rapida de cada uma!


Tags Função
"DOCTYPE html" Informa ao navegador a versão do html que está sendo utilizado.
"html" Elemento basica da estrutura html, ele é quem conterá todos os elementos do seu documento.
"head" Essa tag possui algumas outros tags "css" que editam cores e funcionalidades da pagina,
além de possuir outras tags, como title, usado para dar nome a pagina,
por exemplo. Também utilizado pra fazer o cabeçalho da pagina.
"body" Tal tag é a tag que mostrará o principal da pagina, ela é quem vai apresentar basicamente todo o conteudo visivel da pagina.
"title" Essa tag define o titulo da sua pagina, ou seja, o que será escrito na aba do navegador.
"p" "p" nos dá a opção de escrever um paragrafo.
"h1, h2, h3, h4, h5" Essas tags serão utilizadas para apresentação de titulos, cada numero seguido
do "h", que sobe, o tamanho do titulo diminui.
"ul" Cria uma lista pontual, exemplo:
  • Exemplo 1
"ol" Cria uma lista ordenada, ou seja, coloca numeros em sequencia sozinha, exemplo:
  1. Exemplo 1
"li" É a tag utilizada pra escrita de algo nas listas.
"strong" Utilizado pra deixar as letras em negrito.
"i" Usado para deixar as letras em formatação em italico.
"table" Criação de uma tabela.
"img" É uma tag que não tem fechamento, essa tag tem o funcionamento de colocar uma imagem em sua pagina.
